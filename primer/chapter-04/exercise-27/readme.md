Exercise 4.27
=============

What is the result of each of these expressions?

    unsigned long ul1 = 3, ul2 = 7;

**(a)** `ul1 & ul2`

      00000000000000000000000000000011
    & 00000000000000000000000000000111
    ----------------------------------
      00000000000000000000000000000011
 
**3**

**(b)** `ul1 | ul2`

      00000000000000000000000000000011
    | 00000000000000000000000000000111
    ----------------------------------
      00000000000000000000000000000111

**7**

**(c)** `ul1 && ul2`

**true**

**(d)** `ul1 || ul2`

**true**
